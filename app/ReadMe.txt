Date : 10/12/2019

Auteur : CLEMENT Alexis

Description : Jeu d'obstacles (inspiré du jeu "Flappy Bird") où le but de la partie est de passer au travers des tuyaux le plus longtemps possible sans se faire toucher 

Contrôles :
	- Menu : N'importe quelle touche du clavier (sauf échap) = Accès au jeu
		 Echap = Quitter le jeu

	- Jeu : N'importe quelle touche du clavier (sauf échap) = Sauter
		Echap = Quitter le jeu

	- GameOver : N'importe quelle touche du clavier (sauf échap) = Accès au menu
		     Echap = Quitter le jeu

Bugs connus :
	- Le meilleur score enregistré est remis à 0 quand l'application est quittée

Fonctionnalités du jeu : Le joueur doit :
	- Ne pas se faire toucher par les tuyaux ou par le sol le plus longtemps possible
	- Faire le meilleur score
